# 🚀 HTML Editor AI v5

> **当尖端 AI 遇上原生 Web 开发。** 这不仅仅是一个编辑器，更是您的智能编码副驾驶。

---

## 💎 核心愿景

`HTML Editor AI v5` 是一款专为高效 HTML/CSS 开发而设计的单文件 IDE。它集成了 **Google Gemini 3.1** 强大的生成能力，通过 **JSON Diff 增量更新技术**，让您只需口述指令，即可完成复杂的代码重构。

---

## ✨ 巅峰特性

### 🧠 AI 深度集成

* **智能差分更新**: 不同于传统的全量替换，AI 仅修改代码片段，保持您的注释和结构完整。
* **实时建议系统**: 自动分析代码冗余，提供 UI 优化建议，点击即应用。
* **模型切换**: 支持 `Gemini 3.1 Flash/Pro` 多模型切换，平衡速度与深度。

### 🐙 完美的 GitHub 生态

* **云端同步**: 一键推送 `index.html` 到 GitHub 仓库。
* **Pages 极速部署**: 自动开启静态网页托管，代码改动后实时预览生产环境。
* **智能图床**: 拖拽图片至编辑器，自动上传至 GitHub 仓库并插入全球加速 CDN 链接。

### 🎨 极致的视觉体验

* **算法生成主题**: 按下 `R` 键，基于 HSL 算法生成无穷尽的高对比度极客主题。
* **沉浸式 UI**: 基于毛玻璃效果（Backdrop Blur）的响应式设计，完美适配手机与 PC。
* **结构浏览器**: 树状视图实时解析 DOM 嵌套，代码逻辑一目了然。

## 🛠️ 快捷键指南

| 按键 | 功能描述 |
| --- | --- |

## 🚀 快速开始

1. **下载文件**: 下载并直接在浏览器打开 `index.html`。
2. **配置 AI**:
* 点击左上角 💎 图标。
* 输入您的 `Gemini API Key`。


3. **连接 GitHub**:
* 在底部 GitHub 栏输入 `Personal Access Token`。
* 选择或新建一个仓库。


4. **开始创作**: 在指令框输入 *“帮我做一个酷炫的赛博朋克按钮”*，然后见证奇迹。

## 🧩 技术堆栈

* **Formatter**: [js-beautify](https://github.com/beautify-web/js-beautify)
* **AI Protocol**: Google Gemini API v1beta (JSON Mode)
* **CDN**: jsDelivr (全球图片加速)
